# Game Design Plan

> Synth Town Remake — senior game design review, 2026-03-02.
> Stages 1–9 complete. All audio/UX/accessibility waves applied.

---

## Analysis (10 points)

1. **Нет ритмической воли** — игрок не может задать паузы или акценты. Музыка идёт непрерывно, без синкопации.
2. **Нет перкуссии** — только 5 мелодических зданий, нет drum/noise типов. Ритмический слой отсутствует.
3. **Нет петлевого рекордера** — нельзя записать паттерн и зациклить поверх live. MIDI-экспорт есть, но это не то.
4. **Синдром белого листа** — при старте пустая сетка, нет предзаполненных шаблонов для быстрого старта.
5. **3 типа транспорта, нет экзотики** — нет медленных/диагональных/нерегулярных маршрутов.
6. **Здания не взаимодействуют между собой** — нет резонанса при соседстве одного типа.
7. **Город визуально статичен** — машины едут, но сам город не "дышит" в такт музыке.
8. **Нет ограничений, создающих интересные выборы** — всё строится где угодно, нет зонирования.
9. **Score-display — число, а не ощущение** — нет прогресс-бара к следующему тиру.
10. **Нет shareability** — URL-шаринг работает, но нет "послушать чужой город" при старте.

## Механики

| # | Название | Что добавляет | Сложность |
|---|----------|--------------|-----------|
| M1 | **Drum Building** | Перкуссионное здание (kick/snare/hi-hat через BufferSource). Закрывает главный пробел — бит-мейкинг | mid |
| M2 | **Road Gate** | Знак «пауза»: транспорт ждёт N бит перед проездом. Синкопация и ритмическое программирование без новых зданий | low |
| M3 | **Building Resonance** | 2+ здания одного типа рядом усиливают друг друга (+3 dB + chorus detune). Тактический placement | mid |
| M4 | **Speed Lanes** | Тип дороги Fast/Slow меняет скорость транспорта на участке. Полиритм, разные длительности нот | low |
| M5 | **Night Cycle** | Каждые N тактов день→вечер→ночь. Ночь = reverb+, brightness−. Sonic evolution без действий игрока | mid |
| M6 | **Starter Templates** | 3–4 предзаполненных мини-города на выбор в оверлее при старте. Убирает белый лист | low |
| M7 | **Loop Recorder** | Кнопка Record → слушает 1–4 такта → зацикливает как Audio Buffer поверх live-звука. Layering, живое выступление | high |
| M8 | **Crossfade Districts** | Каждая зона (группа зданий в радиусе 3 тайла) имеет свой мастер-фейдер. DJ-сет из городов | mid |

## Контент

**Метапрогрессия:**
- Unlock-дерево: новые типы зданий открываются за достижения, не только за очки
- «Архитектурные стили» — пресеты визуального скина районов (Industrial / Garden / Neon)

**Новые режимы:**
- **Challenge Mode** — задана мелодия, нужно воспроизвести её постройкой города
- **Minimal Mode** — максимум 8 зданий, 1 маршрут; фокус на тонкой настройке
- **Ambient Mode** — без BPM-сетки, всё случайно и медленно (generative ambient)

**Вариативность:**
- 3 новых типа зданий: Drum, Noise, Arpeggiator (автобег по аккорду)
- 2 новых типа транспорта: Tram (медленный, широкий trigger-радиус), Drone (летит по диагонали)
- Сезонные пресеты эффектов: Spring = bright filter, Winter = heavy reverb

**Поводы возвращаться:**
- Daily Challenge — случайная стартовая конфигурация + цель по score
- «Город недели» — расшаренный URL в оверлее при старте
- Achievement badges, видимые в share-ссылке

## Дизайн

**UI улучшения:**
- Score-бар как горизонтальный fill вместо числа — визуальный прогресс до следующего тира
- Waveform preview в иконке здания в тулбаре (мини SVG синусоиды, живая анимация)
- Контекстное меню правого клика на тайл: Delete / Upgrade / Copy — вместо отдельной панели

**Визуальные усиления:**
- Пульс здания в такт BPM: `scale(1.0 → 1.05)`, синхронизован с beat-dot
- Particle trail у транспорта: car = искры, bicycle = ноты ♩, bus = волны
- Плавный zoom in/out (колесо мыши) с сохранением центра

**Саунд-фидбек:**
- Undo/Redo — обратный звук (reverse плэйбэк последнего триггера)
- Удаление знака → отдельный «whoosh» звук

**Quick wins (< 1 дня каждый):**
- Кнопка «Randomize Notes» в property panel — перетасовывает питч всех зданий в гамме
- Tap Tempo кнопка рядом со слайдером BPM
- Fullscreen toggle (клавиша F или кнопка в транспорте)

## Roadmap

### High Impact
1. **M6 — Starter Templates** — убирает барьер входа, мгновенный эффект на онбординг
2. **M1 — Drum Building** — закрывает главный пробел (перкуссия)
3. **M2 — Road Gate / Pause Sign** — ритмическая воля без архитектурных изменений
4. **Quick wins** — Randomize Notes + Tap Tempo + Fullscreen
5. **Score-бар как fill** + пульс зданий в такт BPM

### Medium
6. **M4 — Speed Lanes**
7. **M3 — Building Resonance**
8. **M5 — Night Cycle** (визуальный + аудио)
9. Waveform-иконки в тулбаре (SVG анимация)
10. Particle trails у транспорта
11. **M7 — Loop Recorder** (первый прототип: запись 2 тактов)
12. Challenge Mode (мелодия-цель)

### Experimental
13. **M8 — Crossfade Districts**
14. Tram + Drone транспорт
15. Arpeggiator-здание
16. «Город недели» / social sharing
17. Ambient Mode без BPM-сетки
18. Кнопка «Generate City» (процедурная генерация)

---

## Фокус первой волны

**M6 + M1 + M2 + Quick Wins.**

Это меняет первые 90 секунд игры (онбординг) и добавляет ритмическую глубину
без изменений в архитектуре модулей.
