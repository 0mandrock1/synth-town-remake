# Manual Extension Area

Папка для ручных, неавтоматизированных дополнений проекта.

## Структура
- `icons/` - кастомные иконки (SVG/PNG), дорисованные руками.
- `overrides/` - локальные наброски/патчи ассетов, которые не относятся к core.
- `references/` - визуальные референсы, moodboard, скетчи.

## Правила
- Сохраняй понятные имена файлов (`transport-bus-v1.svg`, `toolbar-road-icon.png`).
- Не изменяй контракты и тесты ради ассетов.
- Если ассет влияет на поведение, добавь заметку в `docs/development/WORKFLOW.md`.
