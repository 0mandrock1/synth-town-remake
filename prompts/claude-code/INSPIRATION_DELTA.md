# Inspiration Delta (Synth Town Build)

Источник вдохновения: `https://synth.town`.
Файл фиксирует, что именно берём из публичного образа Synth Town для ремейка и как это переводится в задачи разработки.

## Что перенимаем
- Музыкальная читаемость города: визуальные формы объектов должны объяснять звук.
- Мгновенный loop feedback: игрок быстро слышит результат строительства.
- Порог входа низкий: старт через одну кнопку и короткие подсказки.
- Инструментальность: интерфейс ощущается как DAW-игрушка, а не только симулятор.

## Что не копируем напрямую
- Не уходим в тяжёлый 3D/WebGL стек.
- Не добавляем backend/social фичи до client-only релиза.
- Не ломаем ограничение одного `index.html`.

## Обязательные переносы в реализацию
- Shape-to-sound (форма здания = waveform) закрепляется в `prompts/claude-code/SPEC.md`.
- Vehicle-to-envelope (тип транспорта = характер ноты) закрепляется в `prompts/claude-code/SPEC.md`.
- Онбординг первого запуска закрепляется как UX-must в спецификации.

## Quality Gate
Сборка не считается готовой, если одновременно не выполнены 3 условия:
1. Первый музыкальный паттерн собирается за 60-90 секунд.
2. Игрок без подсказки различает минимум 3 типа зданий по форме и звуку.
3. Игрок слышит разницу между `car`, `bicycle`, `bus` при одинаковом маршруте.
